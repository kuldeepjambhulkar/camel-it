# camel-it
NPM package to turn a string into camel case!

## Usage

- npm i camel-it  
- const camelIt = require('camel-it');  
- camelIt(yourStringHere);  

## Example
```javascript
const camelIt = require('camel-it');
console.log(camelIt('hey there this is a normal string'));
// heyThereThisIsANormalString
```
